Creating a new branch
    git switch -c BRANCH_NAME
    [When you create a branch, you automatically switch to that branch]


Fetching list of branches
    git branch


[Note: To know which branch is active, say "git status"]


Switching to another branch
    git switch BRANCH_NAME


Merging a branch with another
    Switch to the master branch and say "git merge BRANCH_NAME" 

Deleting a branch
    git branch -b BRANCH_NAME
